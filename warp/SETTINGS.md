# Warp 手動設定メモ

Settings Sync では同期されないため、新 PC セットアップ時に手動で設定する。

## Appearance

| 項目 | 値 | 場所 |
|---|---|---|
| Theme | CyberWave | Settings > Appearance > Theme |
| Font Size | 14 | Settings > Appearance > Text |
| Window Opacity | 50% | Settings > Appearance > Window |

## Features

| 項目 | 値 | 場所 |
|---|---|---|
| Input Mode | Pinned to Bottom | Settings > Features > Session |

## キーバインド

`keybindings.yaml` で管理（dotfiles）。

```yaml
"editor_view:insert_autosuggestion": tab
"input:open_completion_suggestions": ctrl-space
```
