# VSCode / Cursor 拡張機能リスト

個人のお気に入り拡張機能一覧。  
新しい環境のセットアップ時に参照する。

## 現在の拡張機能を確認

```bash
code --list-extensions
cursor --list-extensions
```



## 共通（どのプロジェクトでも入れる）

| 拡張機能 | ID | 用途 |
|---|---|---|
| Code Spell Checker | `streetsidesoftware.code-spell-checker` | スペルチェック |
| DotENV | `mikestead.dotenv` | .envファイルのシンタックスハイライト |
| EditorConfig | `editorconfig.editorconfig` | .editorconfigの読み込み |
| Git Graph | `mhutchie.git-graph` | Gitのグラフ表示 |
| GitLens | `eamodio.gitlens` | Git操作の強化 |
| Japanese Language Pack | `ms-ceintl.vscode-language-pack-ja` | UI日本語化 |
| Regex Previewer | `chrmarti.regex` | 正規表現のリアルタイムプレビュー |
